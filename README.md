# learnES6
从0.5到1的ES6学习
内容基于B站小马哥老师的ES6讲解课程，
[小马哥-ES6入门到精通](https://www.bilibili.com/video/BV1ay4y1r78B/?spm_id_from=333.337.search-card.all.click)
在老师的基础上进行一些个人的记录和补充，主要是把之前工作中零散的ES6知识串联起来，做一些系统性的学习
对工作中没有使用到，或者没有进行深入使用的特性进行学习

学习的day2发现老师的讲解存在一部分的省略或模糊，会导致学习中的疑惑，此处给出阮一峰老师的相关学习文档网站，方便对照学习。
[JavaScript教程](https://wangdoc.com/javascript/basic/)
[ES6标准入门教程-第三版](https://www.bookstack.cn/read/es6-3rd/sidebar.md)
